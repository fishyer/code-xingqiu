您可以使用以下正则表达式来匹配该字符串：

```scss
(?<!\S)\[语画界XIAOYU\]\s+Vol\.\d+\s+\S+(?!\S)
```

这个正则表达式的含义是：

*   `(?<!\S)`：前面不能有非空白字符。
*   `\[`：匹配 `[` 字符。
*   `语画界XIAOYU`：匹配该字符串。
*   `\]`：匹配 `]` 字符。
*   `\s+`：匹配一个或多个空白字符。
*   `Vol\.\d+`：匹配 `Vol.` 后面跟着一个或多个数字。
*   `\s+`：匹配一个或多个空白字符。
*   `\S+`：匹配一个或多个非空白字符。
*   `(?!\S)`：后面不能有非空白字符。

这个正则表达式使用了“前后断言”，确保匹配到的字符串前后没有非空白字符。同时，使用了 `\s+` 来匹配中间的空格。
